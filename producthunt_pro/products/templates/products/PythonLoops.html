Loops In Python

Python programming language provides the following types of loops to handle looping requirements.

1. While Loop in Python

In python, a while loop is used to execute a block of statements repeatedly until a given condition is satisfied. And when the condition becomes false, the line immediately after the loop in the program is executed.

Syntax 
        
            while expression:
                statement(s)
            # Python uses indentation as its method of grouping statements.
Example of Python While Loop 
        
            a = 0
            while (a < 3):
                a = a + 1
                print("Hi")
            #Output: Hi 
                     Hi
                     Hi
While Loop with else Condition 
    
            a = 0
            while (a < 3):
                a = a + 1
                print("Hi")
            else:
                print("In Else")
            #Output: Hi 
                     Hi
                     Hi
                     In Else
Infinite While Loop in Python 
    
        count = 0
        while (count == 0):
            print("Hello Geek")
        #It is not Recommended to use Infinite Loop as you may need to forcefully terminate the compiler

2. For Loop in Python

In Python, a for loop is used to iterate over a sequence, which can be a list, tuple, string, dictionary, or any other iterable object. The for loop allows you to perform a set of operations on each item in the sequence.

Syntax 
        
            for var in sequence:
                statements(s)

Here are some examples of using for loops in Python:

1. Looping through a List: 
        
            fruits = ["apple", "banana", "cherry"]
            for fruit in fruits:
                print(fruit)
            
            # Output:apple
                     banana
                     cherry
2. Looping through a String: 
        
            text = "Hello"
            for char in text:
                print(char)
            # Output:H
                     e
                     l
                     l
                     o
3. Looping through a Range: 
        
            # You can also use the range() function to create a sequence of numbers and loop through it.
            for i in range(5):  # Loops from 0 to 4
                print(i)
            # Output:0
                     1
                     2
                     3
                     4
4. Looping through a Dictionary: 
        
            # When iterating through a dictionary, you can access both keys and values using the .items() method.
            person = {"name": "John", "age": 30, "city": "New York"}
            for key, value in person.items():
                print(f"{key}: {value}")
            # Output:name:John
                     age:30
                     city:New York
5. Nested for Loops: 
        
            # You can nest for loops to loop through multiple sequences, creating a loop within a loop.
            for i in range(2):
                for j in range(2):
                    print(f"({i}, {j})")
            # Output:(0,0)
                     (0,1)
                     (1,0)
                     (1,1)

Loop Control Statements

In Python, loop control statements allow you to control the flow of loops (e.g., for and while loops). There are three main loop control statements:

1. break Statement

The break statement is used to exit a loop prematurely. When a break statement is encountered within a loop, it immediately terminates the loop, and control is passed to the code following the loop. This is useful when you want to exit a loop based on a certain condition.

Example 
                
                    for number in range(1, 11):
                        if number == 5:
                            break  # Exit the loop when number is 5
                        print(number)
                    # Output:1
                             2
                             3
                             4
2. continue Statement:

The continue statement is used to skip the current iteration of a loop and move on to the next iteration. It is often used when you want to avoid executing some code in a particular iteration but continue with the loop.

Example 
                
                    for number in range(1, 6):
                        if number == 3:
                            continue  # Skip number 3
                        print(number)
                    # Output:1
                             2
                             4
                             5
3. pass Statement:

The pass statement is a placeholder statement that does nothing. It is often used when a statement is syntactically required but you don't want to execute any code. It is commonly used when you are designing a function or class and want to add the implementation later.

Example 
                
                    for letter in 'geeksforgeeks':
                        pass
                    print('Last Letter :', letter)

                    # Output: s
