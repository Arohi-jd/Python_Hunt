What are Sets in Python ?


A set is an unordered collection of unique elements. It is defined by placing a comma-separated sequence of elements inside curly braces {}.

Sets are similar to lists or dictionaries, but they cannot have duplicate elements. The order of elements in a set is undefined, and sets do not support indexing or slicing operations.

Features Of Dictionaries

Sets are one of the most important data structures of python

The features are liste here ➡️.

1️⃣ Mutability

Sets are mutable, meaning you can add and remove elements after the set is created.This eases the user to work with the sets.

2️⃣ Dynamic Size:

Sets can grow or shrink in size as elements are added or removed. There is no need to preallocate space for a set which avoid memory wastage.

3️⃣ Comprehension

Similar to list comprehensions, set comprehensions allow concise creation of sets using a single line of code.

4️⃣ Unique Elements

Sets cannot contain duplicate elements. If you try to add an element that is already in the set, it won't create a duplicate.

Operations on Sets

1️⃣ Creating a Set:


A set can be created using the following syntax :

my_set = {1, 2, 3}

2️⃣ Adding Elements:


A element can be be added to a set using the following syntax :

my_set.add(4)

3️⃣ Removing Elements:


A element can be be deleted from a set using the following syntax :

my_set.remove(2)

4️⃣ Checking Membership:


A element in a set can be verifed using the following syntax :

print(3 in my_set)
