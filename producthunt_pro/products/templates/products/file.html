File Handling

File handling in Python is accomplished through built-in functions and libraries. Python provides a straightforward and powerful way to work with files

Opening and Closing of File

To open a file in Python, you can use the built-in open() function. It takes two arguments: the file's path and the mode in which the file should be opened. The basic modes include:

  • "r":Read (default mode).
  • "w":Write (creates a new file or overwrites an existing file).
  • "a":Append (writes to the end of an existing file).
  • "b:Binary mode.
  • "t:Text mode (default).

Example of opening a file for reading:

        
            file = open("example.txt", "r")
            # perform the operation
            # After you're done with the file, it's crucial to close it using the close() method:
            file.close()

Reading from Files

You can read the content of a file using methods like read(), readline(), or readlines().

Example of reading the entire file:

  • Using read() function: 
                        
                        with open("example.txt", "r") as file:
                            content = file.read()
                            print(content)
  • Using readline() function:

    The readline() method reads and returns one line from the file each time it's called. You can use a loop to read lines one by one until the end of the file is reached.

                        
                        with open("example.txt", "r") as file:
                            line = file.readline()
                            while line:
                                print(line, end='')  # Use end='' to avoid extra newlines
                                line = file.readline()
                        #In this example, readline() is used to read and print lines from the file "example.txt" until there are no more lines to read.
  • Using readlines() function:

    The readlines() method reads all the lines from a file and returns them as a list of strings, with each string representing a line of text.

                        
                        with open("example.txt", "r") as file:
                            lines = file.readlines()
                            for line in lines:
                                print(line, end='')

Writing to files

To write to a file, open it in write or append mode

  • Using write() function:

    The write() method is used to write a single string to a file. You can open the file in write mode ("w") or append mode ("a") and then use write() to add content to the file.

                    
                    with open("example.txt", "w") as file:
                        file.write("This is a line of text.\n")
                        file.write("This is another line of text.\n")
  • Using writelines():

    The writelines() method is used to write a list of strings to a file. Each string in the list represents a line of text.

                    
                    lines_to_write = ["Line 1\n", "Line 2\n", "Line 3\n"]
                    with open("example.txt", "w") as file:
                        file.writelines(lines_to_write)

Appending to file

If you want to append data to an existing file, you should open the file in append mode ("a") and use the write() method or writelines() method

Example:

            
                with open("example.txt", "a") as file:
                    file.write("This is additional content.\n")
                # In this example, we open the file "example.txt" in append mode and use the write() method to add new content to the end of the file.

File Modes

File modes in Python determine the purpose and permissions of opening a file

Common File Modes:

  1. "r": Read (default mode)
    • Opens the file for reading
    • Raises an error if the file doesn't exist.
    • Example: 
                              
                            with open("example.txt", "r") as file:
                                content = file.read()
  2. "w": Write
    • Opens the file for writing.
    • Creates a new file if it doesn't exist.
    • Overwrites the file if it already exists.
    • Example: 
                          
                        with open("output.txt", "w") as file:
                            file.write("This is a new file.")
  3. "a": append
    • Opens the file for writing, but does not overwrite existing content.
    • Creates a new file if it doesn't exist.
    • Appends data to the end of the file.
    • Example: 
                          
                        with open("existing.txt", "a") as file:
                            file.write("This content will be appended")
  4. "b": Binary
    • Used in combination with other modes to open a file in binary mode. For example, "rb" for reading a binary file.
    • Binary mode is typically used when working with binary files, such as images, audio, or other non-text files. In binary mode, data is read and written as raw bytes.
    • Example: 
                              
                            # Reading a Binary File:
                            with open("image.jpg", "rb") as file:
                                binary_data=file.read()
                            
                            # Writing to a Binary File
                            binary_data = b'\x48\x45\x4C\x4C\x4F'  # Binary data (e.g., "HELLO" in ASCII)
                                with open("binary_data.bin", "wb") as file:
                                    file.write(binary_data)

                            # Binary data to append (in this case, some bytes representing binary data)
                            binary_data_to_append = b'\x01\x02\x03\x04\x05'   
                            # Open the binary file in append mode ("ab")
                            with open("binary_data.bin", "ab") as file:
                                # Append the binary data to the end of the file
                                file.write(binary_data_to_append)
                                    
                            print("Binary data appended successfully.")
  5. "x": Exclusive creation
    • Opens a file for exclusive creation, meaning it will fail if the file already exists.
    • Example: 
                          
                        try:
                            with open("new_file.txt", "x") as file:
                                file.write("This is a new file.")
                        except FileExistsError: 
                            print("The file already exists.")

  1. name attribute: This attribute contains the name of file,including the path, that you have opened
  2. mode attribute:This attribute contains the mode in which the file was opened, indicating whether the file is opened for reading, writing, appending, etc.

Navigating Within Files

In Python, you can use the seek() and tell() methods to navigate within a file. seek() is used to move the file pointer to a specific position within the file, and tell() is used to retrieve the current file pointer position. Here's how you can use these methods:

Using seek() and tell() functions: 
            
                # Open a file in read mode
                with open("example.txt", "r") as file:
                    content = file.read()
                    print("Initial content:")
                    print(content)

                    # Use the 'tell()' method to get the current position of the file pointer
                    position = file.tell()
                    print("Current position:", position)

                    # Move the file pointer to a specific position using 'seek()'
                    file.seek(10)  # Move to the 10th byte (position is 0-based)

                    # Read from the new position
                    updated_content = file.read()
                    print("Updated content:")
                    print(updated_content)

                    # Get the new position using 'tell()'
                    new_position = file.tell()
                    print("New position:", new_position)

In this example:

  1. We initially read the entire content of the file and print it.
  2. We use the tell() method to get the current position of the file pointer, which is initially at the end of the file.
  3. We then use the seek() method to move the file pointer to the 10th byte (position is 0-based).
  4. Finally, we use tell() again to get the new position of the file pointer.

Error Handling with Files

Error handling when working with files in Python is important to ensure that your program gracefully handles exceptions that might occur when performing file operations. Some common exceptions you might encounter when working with files include:

  1. FileNotFoundError:Raised when attempting to open or manipulate a file that doesn't exist.
  2. PermissionError:Raised when the program doesn't have the necessary permissions to access the file.
  3. IOError:A generic exception for I/O-related errors.
  4. OSError:A generic exception for operating system-related errors.
  5. Example: 
                        
                        try:
                            with open("example.txt", "r") as file:
                                content = file.read()
                            print("File content:", content)
                        except FileNotFoundError:
                            print("The file does not exist.")
                        except PermissionError:
                            print("You don't have permission to access the file.")
                        except IOError as e:
                            print(f"An I/O error occurred: {e}")
                        except OSError as e:
                            print(f"An operating system error occurred: {e}")
                        except Exception as e:
                            print(f"An unexpected error occurred: {e}")
                        else:
                            print("File was read successfully.")
                        finally:
                            print("File handling complete.")

Using With Statement:

Automatic file closing is achieved in Python using the with statement (context manager). When you open a file using the with statement, the file is automatically closed when the block of code is exited. This ensures that you don't have to explicitly call file.close() to close the file, and it helps prevent resource leaks and potential issues associated with forgetting to close a file.

            
                # Open a file using 'with' for automatic closing
                with open("example.txt", "r") as file:
                    content = file.read()
                    print("File content:", content)

                # The file is automatically closed when exiting the 'with' block
                # You don't need to explicitly call 'file.close()'
                #In the code above, the file "example.txt" is opened for reading within the with block, and when the block is exited, the file is automatically closed.

                Using the with statement is considered best practice when working with files in Python because it ensures proper file closure and helps maintain code readability and cleanliness.

Working with Text and Binary Files

Working with text and binary files in Python involves using different modes and methods to read from and write to these types of files. Text files are human-readable files, often composed of text characters, while binary files contain raw data and are not human-readable. Here's how to work with both types of files:

Working with Text Files:
  1. Opening Text File: To open a text file for reading, you can use "r" mode. To open a text file for writing, you can use "w" mode. For appending to an existing text file, use "a" mode. 
                        
                        # Reading from a text file
                        with open("textfile.txt", "r") as file:
                            content = file.read()
                        
                        # Writing to a text file
                        with open("output.txt", "w") as file:
                            file.write("This is a new text file.")
  2. Reading Text Lines:
    You can read text lines one at a time or as a list of lines using methods like readline() and readlines(): 
                        
                        with open("textfile.txt", "r") as file:
                        line = file.readline()  # Read one line at a time
                        lines = file.readlines()  # Read all lines into a list
Working with Binary Files:
  1. Opening Binary Files:
    To open a binary file for reading, use "rb" mode. For writing to a binary file, use "wb" mode. For appending to an existing binary file, use "ab" mode. 
                        
                        # Reading from a binary file
                        with open("binaryfile.bin", "rb") as file:
                            binary_data = file.read()
                        
                        # Writing to a binary file
                        binary_data = b'\x01\x02\x03'  # Binary data
                        with open("output.bin", "wb") as file:
                            file.write(binary_data)
  2. Writing to Binary File:
    To write data to a binary file in Python, you need to open the file in binary write mode ("wb") and then use methods like write() to write the binary data
                        
                        # Binary data to write (as bytes)
                        binary_data = b'\x01\x02\x03\x04\x05'
                        
                        # Open a binary file for writing
                        with open("output_binary.bin", "wb") as file:
                            # Write the binary data to the file
                            file.write(binary_data)
                        
                        print("Binary data written to the file.")

Working with XML File

Python's built-in xml library allows you to work with XML files.

  1. Reading an XML File:

    Suppose you have an XML file named "data.xml" with the following content

                    
                    <root>
                        <person> 
                            <name> XYZ </name>
                            <age>  18 </age>
                        </person>
                    </root>  
    
                    # You can read and parse it like this:
                    import xml.etree.ElementTree as ET

                    # Parse an XML file
                    tree = ET.parse("data.xml")
                    root = tree.getroot()
                    
                    # Access XML elements
                    name = root.find("person/name").text
                    age = root.find("person/age").text
                    
                    print("Name:", name)
                    print("Age:", age)
  2. Writing to XML File: 
                        
                        import xml.etree.ElementTree as ET

                        # Create an XML structure
                        root = ET.Element("root")
                        person = ET.SubElement(root, "person")
                        name = ET.SubElement(person, "name")
                        name.text = "Bob"
                        age = ET.SubElement(person, "age")
                        age.text = "30"
                        
                        # Create an ElementTree
                        tree = ET.ElementTree(root)
                        
                        # Save the XML to a file
                        tree.write("output.xml")

Managing Directories

The os module in Python provides functions for working with directories and file paths, allowing you to manage directories, create, remove, and list files and directories, and navigate the file system. Here are some common operations for managing directories using the os module:

  1. Getting The Current Working Directory:

    You can use the os.getcwd() function to get the current working directory. 

                        
                        import os

                        current_directory = os.getcwd()
                        print("Current Directory:", current_directory)
  2. Changing the Current Working Directory:

    You can use the os.chdir() function to change the current working directory to a specified path.

                    
                    import os

                    new_directory = "/path/to/new/directory"
                    os.chdir(new_directory)
  3. Creating a Directory:

    Use the os.mkdir() function to create a new directory.

                    
                    import os

                    new_directory = "my_new_directory"
                    os.mkdir(new_directory)
  4. Creating a Directory and Parent Directories:

    You can use os.makedirs() to create a directory along with any necessary parent directories.

                
                import os

                new_directory = "parent/child/grandchild"
                os.makedirs(new_directory)
  5. Removing a Directory:

    To remove an empty directory, use os.rmdir(). To remove a directory and its contents, use shutil.rmtree() from the shutil module.

            
            import os

            directory_to_remove = "directory_to_remove"
            os.rmdir(directory_to_remove)
  6. Listing Files and Directories in a Directory:

    You can use os.listdir() to get a list of files and directories in a specified directory.

            
            import os

            directory_path = "my_directory"
            contents = os.listdir(directory_path)
            print("Contents:", contents)
  7. Checking if a Path Exists:

    You can check if a file or directory exists using os.path.exists().

            
            import os

            path_to_check = "file_or_directory_path"
            if os.path.exists(path_to_check):
                print("Path exists.")
            else:
                print("Path does not exist.")
  8. Checking if a Path Is a Directory or File:

    You can check if a path is a directory or a file using os.path.isdir() and os.path.isfile().

            
            import os

            path_to_check = "directory_or_file_path"
            if os.path.isdir(path_to_check):
                print("It's a directory.")
            elif os.path.isfile(path_to_check):
                print("It's a file.")
            else:
                print("It's neither a directory nor a file.")

File Metadata

The os.path module in Python provides a variety of functions for retrieving metadata and information about files and directories, such as file size, file creation/modification times, checking if a path exists, and more. Here are some of the common functions and methods from os.path for file-related information:

  1. os.path.exists(path):
    • Checks if the specified path (file or directory) exists.
    • Returns True if the path exists, and False otherwise.
      •                         
                            import os

                            path = "example.txt"
                            if os.path.exists(path):
                                print(f"{path} exists.")
                            else:
                                print(f"{path} does not exist.")
  2. os.path.isfile(path):
    • Checks if the specified path is a file.
    • Returns True if the path is a file, and False otherwise.
      •                         
                            import os

                            file_path = "example.txt"
                            if os.path.isfile(file_path):
                                print(f"{file_path} is a file.")
                            else:
                                print(f"{file_path} is not a file.")
  3. os.path.isdir(path):
    • Checks if the specified path is a directory.
    • Returns True if the path is a directory, and False otherwise.
      •                     
                        import os

                        directory_path = "my_directory"
                        if os.path.isdir(directory_path):
                            print(f"{directory_path} is a directory.")
                        else:
                            print(f"{directory_path} is not a directory.")
  4. os.path.getsize(path):
    • Returns the size of the file in bytes.
    • If the path is a directory, it returns the size of the directory itself, not the total size of its contents.
      •                 
                    import os

                    file_path = "example.txt"
                    file_size = os.path.getsize(file_path)
                    print(f"Size of {file_path}: {file_size} bytes")
  5. os.path.getmtime(path):

    Returns the last modification time of the file or directory as a timestamp (seconds since the epoch).

                    
                    import os
                    import datetime
                    
                    path = "example.txt"
                    modification_time = os.path.getmtime(path)
                    formatted_time = datetime.datetime.fromtimestamp(modification_time).strftime('%Y-%m-%d %H:%M:%S')
                    print(f"Last modification time of {path}: {formatted_time}")
